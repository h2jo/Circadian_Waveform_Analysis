Codes for analyzing the data of CCA1 in the Ostreococcus circadian system

"get_preprocess.py"

: This python file reads the data files for the curves in pixels and transforms them into the real-valued curves as well as their splined curves. For the details, see the comments in the python file.

"get_analysis.py"

: This python file reads the data files for x(t) and r(t) to calculate the local cost and global cost. For the details, see the comments in the python file.
